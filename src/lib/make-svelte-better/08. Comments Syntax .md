## Comments Syntax

The problem is that you can't commet a comment. You can't write a comment inside another comment.

The solution is to use JS comment syntax everywhere. You can see an example below.

In HTML:

```html
<!-- <p>
	<!-- <span></span> -->
</p> -->

<!--
<!---->
-->
```

In CSS:

```css
/* div {
	/* display: flex; */
} */

/*
/**/
*/
```

In JS:

```js
/* function add() {
	/* console.log() */
} */

/*
/**/
*/

// function add() {
// 	// console.log()
// }

//
// 	//
//
```
