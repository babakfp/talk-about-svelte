## Comments Syntax

The problem is that you can't commet a comment. You can't write comment inside another comment.

In HTML:

```html
<!--
<!---->
-->
```

In CSS:

```css
/*
/**/
*/
```

In JS:

```js
/*
/**/
*/
```

### How to fix?

In SCSS:

```scss
// // Comment
// .class {}

// Same problem
/*
/**/
*/

// /*
// /**/
// */
```
