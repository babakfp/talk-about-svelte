## Logic Blocks

- Better consistency with `{@html}` and the other blocks.
- A *simple*, *beginner friendly* and *memorable* syntax without `#`, `:`, `/`.

```
{@if}
{@else if}
{@else}
{@endif}
```

Do the same for the other blocks too (`await` ,`then`, `catch`, `endawait`), (`each`, `endeach`) and etc.
