## Code Blocks In HTML

### If Block

No `#`, `:`, `/`.

```
{@if <>}
{@else if <>}
{@else}
{@endend}
```

### Each Block

No `#`, `:`, `/`.

```
{@each <> as <>}
{@endeach}

{@each <> as <>, i}
{@endeach}
```

### keyed Each Block

It's better to change the name to `key each`, because "keyed each" sounds alien.

### Await Block

No `#`, `:`, `/`.

```
{@await <>}
{@then <>}
{@catch <>}
{@endawait}
```
