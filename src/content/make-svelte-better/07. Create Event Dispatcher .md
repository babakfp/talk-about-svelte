## Create event dispatcher

[Dispatch vs Send][dispatch-vs-send]

The word "dispatcher" isn't a familiar and known word. Let's add an explanation in docs that says it's the meaning is the same as the "send" word. Or just change the name:

```js
// Before:
const dispatch = createEventDispatcher()

// After:
const send = createEventSender()
const forward = createEventForwarder()
```
